> **"주식 공부, 이제 스마트하게!"**
> 초보 투자자를 위한 체계적인 금융 교육 앱

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **앱명** | Stocker (스토커) |

## 핵심 기능 (5대 기능)

### 1. 교육 시스템 (Education)
- 챕터별 체계적인 주식 이론 학습
- 단계별 진도 관리 및 북마크
- 진행률 시각화

### 2. 출석 체크 (Attendance)
- 매일 출석 퀴즈로 학습 동기 부여
- 캘린더 기반 출석 현황 확인
- 연속 출석 트래킹

### 3. 퀴즈 시스템 (Quiz)
- 챕터별 O/X 퀴즈
- 즉각적인 정답 확인 및 해설
- 학습 성취도 측정

### 4. 오답노트 (Wrong Note)
- 틀린 문제 자동 수집
- 챕터별 필터링
- 재학습을 통한 완전 학습 지원

### 5. 투자 성향 분석 (Aptitude)
- 20개 문항 성향 테스트
- 5가지 투자자 유형 분류
- 맞춤형 포트폴리오 추천

## 아키텍처

### Clean Architecture + Feature-Based 구조

```
lib/
├── main.dart                    # 앱 진입점
├── app/                         # 앱 레벨 설정
│   ├── config/                  # 라우터, 테마 설정
│   └── core/                    # 공통 모듈
└── features/                    # 기능별 모듈
    ├── auth/                    # 인증
    ├── education/               # 교육
    ├── attendance/              # 출석
    ├── quiz/                    # 퀴즈
    ├── wrong_note/              # 오답노트
    ├── aptitude/                # 성향분석
    └── mypage/                  # 마이페이지
```

### 각 Feature 내부 구조 (3-Layer)
```
feature/
├── data/           # API 통신, DTO
├── domain/         # Repository, 비즈니스 모델
└── presentation/   # UI, 상태관리
```

## Q&A

### 자주 묻는 질문

**Q: 왜 Flutter를 선택했나요?**
> 하나의 코드베이스로 iOS/Android 동시 개발이 가능하며, 빠른 개발 속도와 네이티브에 가까운 성능을 제공합니다.

**Q: 상태관리로 Riverpod을 선택한 이유는?**
> Provider의 개선 버전으로, 타입 안전성과 테스트 용이성이 뛰어나며, 선언적 코드 작성이 가능합니다.

**Q: 오프라인 모드를 지원하나요?**
> 현재는 온라인 모드만 지원하며, 향후 업데이트에서 오프라인 캐싱을 추가할 예정입니다.
