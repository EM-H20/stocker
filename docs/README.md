## 프로젝트 개요

**Stocker**는 초보 투자자를 위한 **주식 교육 앱**입니다.

### 핵심 기능
- 📚 **교육**: 단계별 주식 이론 학습
- 📅 **출석**: 매일 퀴즈로 학습 동기 부여
- ❓ **퀴즈**: O/X 문제로 이해도 검증
- 📝 **오답노트**: 틀린 문제 복습
- 🧠 **성향 분석**: 투자 유형 분석 및 맞춤 추천
- 📒 **노트**: 학습 내용 정리

### 기술 스택
- **Frontend**: Flutter 3.4+, Riverpod, GoRouter
- **Backend**: Node.js, Express.js
- **Database**: MySQL/PostgreSQL
